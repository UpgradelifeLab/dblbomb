# top_bomber
Это мощный sms bomber у которого в запасе целых 50 сервисов для отправки sms, и 1 сервис на звонок!
# Install
git clone https://github.com/UpgradelifeLab/dblbomb
pip3 -r requestments.txt
python3 dblbomb.py
